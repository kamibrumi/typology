# Designing Example: Embryo Selection Decision Support

This example shows how the decision-making typology can be used to move from a sensitive clinical decision problem to a visualization interface concept. The example is based on an embryo selection scenario in which an embryologist must choose which embryo, or shortlist of embryos, to prioritize for transfer from a current patient’s IVF cycle.

This scenario was inspired by my collaboration with the [Visual Computing Group @ Harvard University](https://vcg.seas.harvard.edu/). In our collaborative paper, [*EmbryoProfiler*](https://ieeexplore.ieee.org/document/11278822), we used the typology of decision-making tasks to describe the embryo selection process and then used that structured decision model to inform the design of a visualization decision-support tool.
