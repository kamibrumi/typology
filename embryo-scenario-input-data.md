The starting point is a dataset of embryo images and associated attributes. For the tutorial, participants should think of the data in two layers.

### Current Patient Cohort

The current patient cohort is the set of embryos being considered for transfer in the present decision. This is the actual option set.

Each row represents one embryo candidate from the current patient’s IVF cycle. In practice, this cohort may contain around 5–9 fertilized eggs or embryos.

### Historical Clinic Dataset

The historical clinic dataset contains embryo images and attributes from previous patients treated at the clinic. These cases are not the current options. Instead, they provide background evidence that can help the embryologist make a more data-driven decision.

### Dataset Columns

Each row in the provided dataset represents one embryo image. Each column represents an image attribute, morphology measure, patient/cycle context attribute, hormone measure, or outcome-related label. (See the main page for the full list.)
